# Vendored emoji

Generated by `yarn sync-emoji` — do not edit by hand.

- `unicode/` — [C1710/blobmoji](https://github.com/C1710/blobmoji) (Apache-2.0),
  Google's original blob designs for standard Unicode emoji.
- `blob/` — the [blobs.gg](https://blobs.gg/) community set via
  [Raymo111/emoji](https://github.com/Raymo111/emoji) (Apache-2.0), the custom
  `:shortcode:` blobs used on Discord.

Only emoji that appear in a post are vendored; the full sets are ~16 MB.
